## 快速开始

### 全量build

​	`sh bin/kylin-cube-full-build.sh cube_name`



### 增量build

- build 1天

  `sh bin/kylin-cube-incr-build-by-1day.sh cube_name '2019-09-01'`

- build 若干天

  `sh bin/kylin-cube-incr-build-by-start-end.sh cube_name '2019-09-01' '2019-09-11'  `
